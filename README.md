# Doki Doki Garrafão Club

> Um visual novel educacional sobre modelagem de banco de dados — onde cada decisão tem consequências.

---

## O que é isso?

**Doki Doki Garrafão Club** é um jogo no estilo visual novel em que você é contratado para criar um sistema de controle de garrafão d'água para uma startup. Ao longo da história, a chefe te apresenta problemas reais de modelagem de banco de dados e você precisa tomar decisões estruturais que vão moldar o destino do sistema — e da sua carreira.

O jogo tem múltiplos finais dependendo das suas escolhas.

---

## Por que foi feito?

O projeto nasceu de uma ideia educacional simples: mostrar para pessoas ainda sem experiência em desenvolvimento que **até decisões bem-intencionadas podem ter um impacto negativo no software**.

Essa é a proposta do jogo: contextualizar erros comuns de modelagem dentro de uma narrativa, para que o aprendizado seja menos abstrato e mais memorável.

## Stack

- React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion
- Vite
- Lucide React

---

## Como rodar

```bash
npm install
npm run dev
```

Acesse em `http://localhost:3000`.

---

## Sobre o desenvolvimento

O projeto foi **vibe codado** — desenvolvido de forma rápida e iterativa, sem arquitetura planejada previamente. Todo o jogo vive em um único arquivo `src/App.tsx`. Não é um exemplo de boas práticas de organização de código, mas é um exemplo de como ideias educacionais podem ganhar vida rapidamente com as ferramentas certas.

O objetivo nunca foi o código em si, mas a experiência que ele proporciona!!!
